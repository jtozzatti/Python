# 🐍 Python Programming Studies

Este repositório reúne exercícios, desafios e implementações desenvolvidas em Python ao longo da graduação e de estudos pessoais.

O foco está em fortalecer a lógica de programação, compreender estruturas da linguagem e escrever código organizado, legível e eficiente.

---

## 🎯 Objetivo

* Consolidar fundamentos da linguagem Python
* Desenvolver raciocínio lógico e resolução de problemas
* Praticar estruturas condicionais, laços e funções
* Organizar código de forma clara e escalável
* Construir uma base sólida para projetos mais avançados

---

## 📂 O que você vai encontrar aqui

* Exercícios de lógica
* Estruturas condicionais e loops
* Funções e modularização
* Manipulação básica de dados
* Pequenos projetos práticos
